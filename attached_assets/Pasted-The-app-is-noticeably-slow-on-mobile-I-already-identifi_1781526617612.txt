- LiveMarketContext → websocket connect
- WatchlistContext → watchlist fetch
- currencyStore.fetchRate()
- brokerStore.loadAccounts()
- dashboard queries

This overloads mobile devices during first paint.

Implement staged loading:

Critical:

- dashboard data
- websocket connection

Deferred (1500ms delay):

- watchlist load
- currencyStore.fetchRate()

...

Do not delay anything required for chart rendering or broker connectivity.
