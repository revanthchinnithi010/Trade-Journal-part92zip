Build a professional Binance-style mobile trading platform using:

- React + Vite
- TailwindCSS
- Lightweight Charts from tradingview.github.io
- Zustand
- Framer Motion
- Lucide React icons

IMPORTANT:
- Use Lightweight Charts ONLY
- Do NOT use TradingView iframe widget
- Do NOT use TradingView embed widget
- Build fully custom UI
- Focus on Binance mobile futures UI style
- Mobile-first and tablet optimized
