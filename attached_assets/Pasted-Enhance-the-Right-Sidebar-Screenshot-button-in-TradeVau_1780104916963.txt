Enhance the Right Sidebar Screenshot button in TradeVault.

==================================================
FEATURE
==================================================

When user clicks the Screenshot button in the right sidebar,
open a professional floating dropdown menu with 2 options:

1. Snapshot
2. Copy Live Chart Link

==================================================

When Snapshot is clicked:

Capture FULL chart area including:
- candles
- indicators
- drawings
- trendlines
- long/short positions
- symbol
- timeframe
- floating elements
- chart overlays
- price scale
- toolbar overlays if visible

IMAGE QUALITY
==================================================

Generate:
- Full HD 1080p image
- High sharpness
- Crisp text rendering
- No blurry scaling
- GPU optimized rendering

Resolution:
1920x1080

Support:
- PNG export
- tablet landscape optimized

SNAPSHOT UI
==================================================

After capture:
show mini popup:

- Preview thumbnail
- Download button
- Copy image button
- Share button

==================================================

Generate a shareable live chart URL including:

- current symbol
- timeframe
- chart type
- indicators
- drawings
- zoom state
- price scale position
- theme mode
